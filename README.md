## EasyTransThem

EasyTransThem是一个基于GO的跨平台文件服务器（类网盘），适用于个人文件的临时传输和存储。

客户端筹备中，实现自动同步和更稳定的传输。

##### 演示

```
Usage:
  -h    Show this help.
  -p    Server port for listening. (default 9569)
  -v    Show version information.
```

```
用法：
  -h    显示帮助
  -p    端口号（默认9569）
  -v    显示版本信息
```
